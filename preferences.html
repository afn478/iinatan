iinatan

Hover subtitles to see dictionary popups in IINA. Japanese is supported by default with Jitendex/Yomitan dictionaries.

Playback

The video pauses while you read. It will not auto-resume when the popup closes.

Language

Language changes parser and candidate behavior, not which dictionaries are enabled. Experimental Latin languages use whole-word lookup; French/German also generate deinflection candidates.

Dictionary Popup

Dictionaries

Recommended for large dictionaries. Imports may take longer.
Use Plugins -> iinatan -> Dictionaries -> Manage Dictionaries... to import, enable, disable, and reorder installed dictionaries.

Appearance
